# Active Directory Certificate Services

## CA Name

ATLAS-ROOT-CA

## Configuration

- Enterprise CA
- Root CA
- RSA 4096-bit Key
- SHA256 Hash Algorithm
- Certification Authority Web Enrollment

## Purpose

Provides a trusted internal PKI infrastructure for:

- VPN Certificates
- User Certificates
- Computer Certificates
- RDP Certificates
- Web Server Certificates

## Verification

- Certification Authority service operational
- Web Enrollment available at:

http://localhost/certsrv
