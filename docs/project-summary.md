# Project Summary

## Objective

Build an enterprise-style infrastructure environment with centralized identity management, network segmentation, policy enforcement, secure remote access, and role-based file services.

## Technologies

- pfSense
- Active Directory
- DNS
- Group Policy
- OpenVPN
- Windows Server
- Ubuntu Server

## Deliverables

- Segmented Network
- Active Directory Design
- Security Groups
- Group Policy Baselines
- Enterprise File Server
- DNS Resolver
- Firewall Policies
- OpenVPN Remote Access
